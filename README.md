## 축구 선수의 리그와 소속 클럽 그리고 포지션에 따른 최고 이적료에 대한 분석

#### 데이터 변수 종류


`name` : 이름 <br>
`full_name` : 전체 이름<br>
`age` : 나이 <br>
`height` : 키 (cm) <br>
`nationality` : 국적 <br>
`place_of_birth` : 출생장소 <br>
`price` : 현재 이적료 <br>
`max_price` : 최대 이적료 <br>
`position` : 포지션 <br>
`shirt_nr` : 등 번호 <br>
`foot` : 주 발 <br>
`club` : 소속 클럽 <br>
`contract_expires` : 계약 종료 <br>
`joined_club` : 계약 시작 <br>
`player_agent` : 에이전트 <br>
`outfitter` : 계약 <br>
`league` : 리그 <br>
